### Available Endpoints

```
# Users

GET /users
POST /users
POST /users/verify?token=
POST /users/authenticate
POST /users/2fa
POST /users/forgot-password
POST /users/reset-password
PUT /users/:id
DELETE /users/:id

# Posts

GET /posts
POST /posts
POST /posts/view
PUT /posts/:id
DELETE /posts/:id
```
