### Available Endpoints

```
# Users

GET /users
GET /users/verify?token=
POST /users
POST /users/authenticate
POST /users/2fa
POST /users/forgot-password
POST /users/reset-password
PUT /users/:id
DELETE /users/:id

# Posts

GET /posts
POST /posts
PUT /posts/:id
DELETE /posts/:id
```
